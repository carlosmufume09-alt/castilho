================================================================================
                      COMO COMEÇAR
================================================================================

1. Instalar dependências:
   $ pnpm install

2. Setup banco de dados:
   $ npx prisma migrate dev --name init
   $ npx ts-node prisma/seed.ts

3. Rodar servidor:
   $ pnpm dev

4. Acessar:
   Vitrine:      http://localhost:3000
   Admin:        http://localhost:3000/login
